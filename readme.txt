== Description ==

My Mitsu, is a webservice in Japan, allows users to create an estimation(calculation) form. It is suited for business persons.

You can put a My Mitsu form in your website, by filling in a html code, like
<iframe src="..." width="..." height="..."></iframe>

This plugin allows you to output an iframe html tag in a simple way.

[mymitsu]274[/mymitsu] will output <iframe src="https://my-mitsu.jp/estimation/274" id="mymitsu" width="640" height="480"></iframe>

Optionally, a shortcode accepts an ID (useful for customizing), width, and height as attributes.

== Installation ==

1. Install a plugin and activate it
1. Upload `shortcode_for_my_mitsu_estimation_form' to the `/wp-content/plugins/` directory
